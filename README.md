# Simon-game

A fun and interactive **Simon Game** built using **HTML**, **CSS**, and **JavaScript**.

---

## Features

- Color sequence memory game
- Interactive button animations
- Level progression
- Game over screen
- Keyboard start support

---

## Technologies Used

* HTML5
* CSS3
* JavaScript

---

## How to Play

1. Press any key to start the game.
2. Watch the color sequence carefully.
3. Repeat the sequence by clicking the buttons.
4. The sequence increases after every level.
5. The game ends if you click the wrong button.
